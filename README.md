# News_gazer
증권 뉴스를 활용해 이슈 종목의 주식 변화를 볼 수 있는 사이트
<br>
2025.1월

<br>

## 목적
SQL + Python 공부

<br>

## 기능
- top10목록 <br>
  기업별, 업종별로 해당 종목이 나온 횟수와 뉴스 링크 등의 정보를 제공한다.
- 상세보기 <br>
  해당 종목의 주식 정보 그래프에 나온 횟수 데이터 결합하여 보여준다.

## 기술 스택
- Python
- Streamlit
- MySQL
- yfinance


## 실행
- python -m pip install -r requirements.txt
- py sql.py
- streamlit run chartF.py
